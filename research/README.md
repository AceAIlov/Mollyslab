# Research — Azumi Cognitive & Emotional Framework

The **research** directory documents the theoretical foundation, experiments, and metrics
behind Azumi’s emotional computation model, which builds upon the **Kimi K2-Instruct engine**.

### Structure
| File | Description |
|------|--------------|
| `emotional_architecture.md` | Breakdown of Azumi’s multi-layered emotional core |
| `kimi_k2_internals.md` | Technical interpretation of Kimi K2-Instruct mechanisms |
| `azumi_whitepaper.md` | Conceptual whitepaper describing emotion as computation |
| `metrics/` | Empirical metrics for evaluating empathy and stability |
